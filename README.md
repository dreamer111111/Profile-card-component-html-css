### What I learned

During this project, I focused heavily on **accessibility** and **modern CSS positioning**.
First time i was creating two background image positiong so it was confusion but i learned this thing properly.

So, I learned how to use the 4-value syntax for `background-position` to place decorative elements relative to the viewport. This ensures the design stays consistent even on large monitors:

```css
body {
  background-position:
    right 52vw bottom 35vh,
    left 48vw top 52vh;
}
```
